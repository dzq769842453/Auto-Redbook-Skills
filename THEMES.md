## 使用方法

在渲染Markdown文件时，可以通过`--style`参数指定使用的主题：

```bash
# 使用默认主题
node scripts/render_xhs_v2.js note.md

# 使用植物风格主题
node scripts/render_xhs_v2.js note.md --style botanical

# 使用终端风格主题
node scripts/render_xhs_v2.js note.md --style terminal
```
